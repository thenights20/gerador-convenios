# Gerador de Convênios em PDF

Site estático para GitHub Pages. O comprovante de inscrição do CNPJ emitido pela Receita Federal é lido localmente no navegador; nenhum arquivo ou dado é enviado para servidor.

## Publicação rápida

1. Crie um repositório público no GitHub.
2. Envie `index.html`, `style.css` e `app.js` para a raiz do repositório.
3. Abra **Settings → Pages**.
4. Em **Build and deployment**, selecione **Deploy from a branch**.
5. Escolha a branch `main`, pasta `/(root)` e clique em **Save**.

O endereço ficará no formato `https://SEU-USUARIO.github.io/NOME-DO-REPOSITORIO/`.
